# AGENTS.md

## Default mode
- Codex handles implementation
- Git/GitHub are the source of truth
- Medium or larger changes should go through branch + PR

## Before changing code
Read:
1. README.md
2. AGENTS.md
3. CONTRIBUTING.md (if present)
4. relevant code and tests

## Rules
- Do not work directly on main/master for meaningful changes
- Do not modify unrelated files
- Always report: what changed, validation, risks, next steps
- Do not claim completion without evidence
